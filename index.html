BUILT TO BE HANDED OVER

Continuity is not an afterthought — it is the design principle

01
One Folder
The entire system lives in a single folder. Copy it to a USB drive or Google Drive. No cloud lock-in, no lost access.
02
One Command
docker compose up -d — the next AGS runs this once and the full system is live, with all data intact.
03
Full Data Export
Built-in backup export from within the system. Every task, document, and minute — archived and portable.
04
Written Guide
Step-by-step setup guide included. A non-technical successor can be fully operational within 30 minutes.
